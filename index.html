How To Use?

Sebelum mulai pake diwajibkan install ekstensi Chrome untuk mengizinkan CORS (karena halaman ini akan melakukan request ke gform) Link Download

Aktifkan ekstensi dengan mengklik tombol extension (icon puzzle) di kanan atas browser, kemudian klik ekstensi Allow CORS. Akan muncul menu dengan huruf C besar, klik huruf tersebut dan jika sudah berwarna artinya ekstensi sudah berjalan

Masukan data sesuai rayon, nomor absen dan tanggal mulai auto input LKH. Pilihan checkbox kelas 10 digunakan untuk input kegiatan pramuka sementara checkbox islam digunakan untuk pengajian malam.
PENTING
DIMOHON UNTUK TIDAK MELAKUKAN SPAM KARENA JIKA TERDETEKSI SPAM KEMUNGKINAN BESAR GFORM LKH AKAN DITAMBAHKAN AUTENTIKASI.
KAMI TIDAK MENYIMPAN DATA DIRI ANDA (KECUALI JIKA ANDA MELAKUKAN SPAM, NO ABSEN ANDA AKAN DI BLACKLIST DARI PENGUNAAN AutoLKH INI.

Jika semua data sudah terisi, klik tombol Gaskeun. Progress auto input LKH bisa dilihat di terminal sebelah kanan.
